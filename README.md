# Tic Tac Toe (Web Application)

#Overview
This project is a browser-based Tic Tac Toe game built using HTML, CSS, and JavaScript. It demonstrates core web development concepts including DOM manipulation, event handling, and interactive UI design.

The game allows two players to play in real-time on the same device with a clean and responsive interface.

---

# Objective
The goal of this project was to:
- Build an interactive web application from scratch  
- Strengthen JavaScript fundamentals (DOM manipulation & logic)  
- Design a simple and user-friendly interface using HTML & CSS  

---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

#Features
- 🎲 Two-player gameplay  
- Real-time board updates using DOM manipulation  
  - Automatic win detection (rows, columns, diagonals)  
- Draw detection  
- Game reset functionality  
-  Clean and responsive UI  

---

#Key Concepts Applied
- DOM manipulation  
- Event listeners  
- Game state management  
- Conditional logic and functions  
- UI updates based on user interaction
